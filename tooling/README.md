# Tooling

Development, diagnostic, and utility binaries for the platform.

## Tools

### extractor — Bundle Extraction
DMG mounting and .app bundle extraction. Used by macrun for application preparation.

### inspector — Bundle Analysis
Static analysis of .app bundles: Info.plist parsing, Mach-O header inspection,
framework probing, and capability assessment.

### benchmark — Performance Measurement
Latency, throughput, and overhead benchmarks for execution strategies.

### devtools — Developer Utilities
Debugging aids, test harnesses, and development workflow tools.
